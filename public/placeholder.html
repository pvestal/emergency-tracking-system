Development Environment

Local Development Setup

This is a placeholder page for the local development server. The full application requires:

  • A built Vue.js application
  • Firebase configuration
  • Proper environment variables

Next Steps

  1. Complete your Firebase configuration in .env.local
  2. Run npm run serve to start the full development server
  3. Or run npm run build followed by npm run dev for an optimized version

Environment Variables

Make sure your .env.local file contains the following Firebase configuration:

VUE_APP_FIREBASE_API_KEY=your-api-key
VUE_APP_FIREBASE_AUTH_DOMAIN=your-project-id.firebaseapp.com
VUE_APP_FIREBASE_PROJECT_ID=your-project-id
VUE_APP_FIREBASE_STORAGE_BUCKET=your-project-id.appspot.com
VUE_APP_FIREBASE_MESSAGING_SENDER_ID=your-messaging-sender-id
VUE_APP_FIREBASE_APP_ID=your-app-id

Firebase Setup

For working with Firebase locally:

  1. Create a Firebase project at console.firebase.google.com
  2. Enable Authentication with Google and GitHub providers
  3. Create a Firestore database with appropriate security rules
  4. Add a web app to your Firebase project to get configuration values
